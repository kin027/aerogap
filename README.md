# Popular Unserved Routes

## Overview
As an avgeek (aviation enthusiast), I'm interested in many different areas of aviation and the airline industry, one of which is passenger flows (especially from my home airports).

This project analyzes 2024 Bureau of Transportation Statistics data (from the U.S. government) to identify the most popular destination airports from a given departure airport that are not served by a nonstop flight

## Data Sources
- BTS T-100 dataset (for finding airport pairs connected with a nonstop flight)
- BTS DB1B dataset (for passenger flow data)

## Tools
- Python
- pandas
- matplotlib
