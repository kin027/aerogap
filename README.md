# Popular Unserved Routes

## Overview
Out of personal curiosity in seeing where people are traveling (especially from my home airports), this project analyzes Bureau of Transportation Statistics data (from the U.S. government) to determine the most popular destination airports from a given departure airport that are not served by a nonstop flight

## Data Sources
- BTS T-100 dataset (for finding airport pairs connected with a nonstop flight)
- BTS DB1B dataset (for passenger flow data)

## Tools
- Python
- pandas
- matplotlib
