# Popular Unserved Routes

## Overview

This program analyzes 2024 Bureau of Transportation Statistics data (from the U.S. government) to identify the most popular destination airports from a given departure airport that are not served by a nonstop flight

I wrote this because I'm an avgeek and I'm also curious to see passenger flows on flights, especially from my home airports :D

## Data Sources
- BTS T-100 dataset (for finding airport pairs connected with a nonstop flight)
- BTS DB1B dataset (for passenger flow data)
